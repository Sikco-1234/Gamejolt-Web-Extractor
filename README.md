HOW TO USE THE TOOL:

For gamejolt games you first need start the project and use inspect element to find the "Index.html" on the page, for other sites its recomended to find the "Index.html" in the network tab
Once it's copied you need to first click on the "Import project from "Index.html"" button, once the process is done, click on the "Extract assets from "Index.html""
And at final once the extraction is done click on the "Compile assets to .ZIP file" button, once the .ZIP is downloaded use the tool Unpackager to convert it to a funcitonal SB3 file
